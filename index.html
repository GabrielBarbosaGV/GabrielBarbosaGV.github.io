HTMX is a HTML extension library that allows for asynchronous operations to alter the DOM.
One possible usage is the following, as shown in the technology's official page: 

  
    
<script src="https://unpkg.com/htmx.org@1.8.6"></script>
<!-- have a button POST a click via AJAX -->
<button hx-post="/clicked" hx-swap="outerHTML">
  Click Me
</button>

This will allow the result of the POST request, to path "/clicked", to swap the content of the button. This HTML extension is great for not overengineering simple web-pages. To be fair, this very page might be overengineered, but I just love SvelteKit that much.
